The pitch

If you have a JavaScript project with dependencies through npm then this is for you!

Npm alerts checks your package.json file nightly to see if a dependency has become out of date and emails you if it has.

That's it! Just sign up above to get started.

About

npm alerts is hosted on Heroku and uses the MEAN-stack (i.e. MongoDB, ExpressJS, AngularJS, and NodeJS. It also uses Twitter Bootstrap and Angular Bootstrap.

If you're interested in helping come to GitHub.
